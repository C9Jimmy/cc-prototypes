About

具備前端開發與 UI/UX 設計的雙重實務背景。曾於醫療軟體公司以 Vue.js 參與 HIS 系統從 table-based 架構重構為模組化元件，並獨立建置 Web App 登入頁面、使用 Git / GitLab 進行版本控管。在益林，我負責六家企業官網的 HTML/CSS 切版，使用 Bootstrap 建置響應式頁面。目前正學習 Vue 3 Composition API 與 TypeScript，並以此網站作為實作驗證。設計師出身的前端視角，能精準理解視覺意圖，有效降低設計工程溝通成本。

面試官，您好！很開心能有機會向您介紹自己。

我是程資閔，畢業於國立虎尾科技大學多媒體設計系。熱衷於持續學習的跨領域工作者，領域涵蓋前端開發、UI/UX 設計與視覺設計。習慣透過廣泛閱讀吸收新知，保持對資訊趨勢的敏銳度，以確保我的實作能跟上快速變化的數位時代。

關於未來規劃

我認為在 AI 世代，技術門檻會降低，但「協作、溝通與人的溫度」反而成為專案成功的關鍵。我擅長擔任設計與工程之間的橋樑，將視覺語言精準轉化為可執行的前端邏輯。希望結合過去在 Vue.js 開發、UI/UX 規劃及跨團隊協作的多元經驗，協助團隊在數位轉型的浪潮中，打造兼具美感與實用性的產品。
